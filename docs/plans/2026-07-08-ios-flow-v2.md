- [ ] B3 Long-press context menu per element type (delete / duplicate /
      edit / layer order); the only delete path; undo toast after delete.
- [ ] B4 Corner-handle resize + two-finger rotate; text typed in place;
      accordion style bar (Aa / color / AI / mic -- one row expanded at a
      time).

### Phase C -- Mira inline card, Keep pattern, working/failure states

- [ ] C1 Inline Mira card above the input bar (no fullscreen chat from the
      canvas); suggestion chips context-aware.
- [ ] C2 Receipt banner: states what changed + what was kept; auto-keeps;
      one-tap revert.
- [ ] C3 Working state: appears only past 400 ms, verb-specific copy in the
      input bar, Stop control, breathing effect on the affected element,
      lock-element-not-screen; 30 s timeout converts to failure.
- [ ] C4 Failure: three responses (clarify / retry / redirect), no red, no
      partial edits, prompt refilled. Wired to chat :8003.
